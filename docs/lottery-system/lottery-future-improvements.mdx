## 개요

1차 개발 이후, 운영 요구사항 및 새로운 비즈니스 요구에 따라 다음과 같은 기능들을 우선순위에 따라 추가로 개발할 필요가 있습니다.

1.  **당첨자 무효화 기능 추가**
2.  **아트센터 예약 추첨 기능 추가**
3.  **블랙리스트 등록/수정 기능 추가**

---

### 1. 당첨자 무효화 기능 추가

SNS 이벤트 등의 경우, 추첨을 실행한 후에 당첨자의 이벤트 조건 충족 여부를 확인하고 일부를 당첨에서 제외해야 하는 경우가 발생합니다. 현재는 추첨 실행 이력 전체를 기준으로 관리되므로, 개별 당첨자 단위로 '무효화' 처리를 할 수 있는 기능이 필요합니다.

### 2. 아트센터 예약 추첨 기능 추가

현재 추첨 시스템을 확장하여, LG 아트센터의 특정 공연이나 전시 예약에 대한 추첨 기능을 추가합니다. 이를 통해 공정하게 예약 기회를 분배할 수 있습니다.

### 3. 블랙리스트 등록/수정 기능 추가

관리자가 어뷰징 사용자 등을 직접 블랙리스트에 추가하거나 기존 정보를 수정할 수 있는 UI 및 API 기능이 필요합니다.
